Wiring for ESP8266

| Serial Cable | ESP8266 |
| ------------- | ------------- |
| BLACK | GND (pin 6)  |
|RED | V+  (pin 4)|
|GREEN | RX (pin 3)|
|WHITE | TX (pin 2)|
